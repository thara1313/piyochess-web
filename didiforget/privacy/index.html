1. 取得・利用するデータ

Didiforget は、利用機能と権限設定に応じて次の情報を取り扱います。

アカウント情報

Google または Apple サインイン時に、uid、表示名、必要に応じてメールアドレスを取り扱います。

写真・カメラ

撮影または選択した写真を持ち物チェックと AI 判定に利用します。撮影画像を自動で端末の写真ライブラリへ保存することはありません。

位置情報

現在地の天気表示、および外出時リマインド通知のために利用します。

通知関連情報

通知許可状態、FCM または APNs トークンを通知配信に利用します。

アプリ内データ

設定、テンプレート、履歴、振り返りメモ、利用回数などをアプリ機能提供のために扱います。

クラウド保存

サインイン時のみ Firestore に保存します。ゲスト利用時はクラウド保存を行いません。

1. Data We Handle

Depending on app features and permission choices, Didiforget may handle the following data.

Account data

When users sign in with Google or Apple, the app may handle their uid, display name, and email address when needed.

Photos and camera

Captured or selected images are used for item checks and AI suggestions. Photos are not automatically saved to the device photo library.

Location

Location is used for local weather display and outing reminder notifications.

Notification-related data

Notification permission status and FCM or APNs tokens are used to deliver notifications.

In-app data

Settings, templates, history, reflection notes, and usage counts are handled to provide the app experience.

Cloud storage

Data is stored in Firestore only when the user signs in. Guest use does not store data in the cloud.

1. 我们处理的数据

Didiforget 会根据用户使用的功能和权限设置，处理以下数据。

账号信息

当用户通过 Google 或 Apple 登录时，应用可能处理 uid、显示名称，以及在必要时处理电子邮箱地址。

照片与相机

拍摄或选择的图片会用于物品检查和 AI 建议。应用不会自动将照片保存到设备相册。

位置信息

位置信息用于显示当地天气，并用于外出提醒通知。

通知相关数据

通知权限状态以及 FCM 或 APNs token 会用于发送通知。

应用内数据

设置、模板、历史记录、回顾笔记和使用次数等数据会用于提供应用功能。

云端存储

只有在用户登录时，数据才会存储到 Firestore。访客模式不会将数据保存在云端。

Third-party disclosure

3. 存储位置与服务提供方

使用的平台

  • Firebase Authentication
  • Cloud Firestore
  • Firebase Cloud Messaging
  • Firebase AI / Gemini 图像与文本处理

向第三方提供

除法律要求或使用受托服务提供方所必需的情况外，用户数据不会提供给第三方。

4. 保存期間

  • アカウント利用中は必要な範囲でデータを保持します。
  • 履歴や設定はユーザー操作で削除できます。
  • アカウント完全削除時は、クラウド保存データと認証アカウントを削除します。

4. Retention

  • Data is retained as needed while the account remains in use.
  • History and settings can be removed by the user.
  • When the account is fully deleted, cloud-stored data and the authentication account are deleted.

4. 保存期限

  • 在账号持续使用期间，数据会在必要范围内保存。
  • 历史记录和设置可由用户自行删除。
  • 当账号被彻底删除时，云端保存的数据和认证账号也会被删除。

5. ユーザーの権利

  • 通知をオフにする
  • 位置情報をオフにする
  • 写真保存をオフにする
  • 履歴を削除する
  • データをエクスポートする
  • アカウントを完全削除する

5. User Controls

  • Turn notifications off
  • Turn location access off
  • Turn photo saving off
  • Delete history
  • Export data
  • Fully delete the account

5. 用户可进行的控制

  • 关闭通知
  • 关闭位置权限
  • 关闭照片保存
  • 删除历史记录
  • 导出数据
  • 彻底删除账号

7. Children

Didiforget is not a children-only app and is not primarily intended for children under 13.

7. 儿童相关说明

8. セキュリティ

  • 通信には Firebase などの提供基盤を利用します。
  • 必要最小限のデータのみを保存します。
  • 通知トークンは画面表示時にマスクして扱うことがあります。

8. Security

  • Network communication uses platforms such as Firebase.
  • Only the minimum data needed for the feature set is stored.
  • Notification tokens may be masked when shown on-screen.

8. 安全性

  • 网络通信使用 Firebase 等提供的平台。
  • 只保存实现功能所需的最少数据。
  • 通知 token 在界面显示时可能会进行遮罩处理。
