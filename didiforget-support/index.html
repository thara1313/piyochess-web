初回案内の流れ

1. カメラ確認を説明

撮影または選択した画像を使って持ち物候補を確認できることを最初に案内します。

2. 通知を先に確認

位置情報の許可前に、「外出した際に通知を受け取りますか」と確認し、忘れ物防止のための通知であることを説明します。

3. 位置情報の用途を限定

現在地の天気表示と、外出時の通知にのみ利用することを説明します。

4. 精度の期待値を共有

自宅から出る時や、一定時間過ごした場所から離れる時を主な通知対象とし、電車など高速移動中は精度に限界があることも伝えます。

First-use guidance

1. Explain the camera check

The app first explains that captured or selected images can be used to review likely belongings.

2. Ask about reminders first

Before requesting location access, the app asks whether the user wants reminder notifications when going out.

3. Limit the reason for location

The app explains that location is used only for local weather and outing reminder notifications.

4. Set realistic expectations

Reminders focus on leaving home or leaving a place after staying there. Accuracy is limited during fast transit such as trains.

首次使用说明

1. 先说明相机检查

应用会先说明，拍摄或选择的图片可以用来确认可能需要携带的物品。

2. 先询问是否需要提醒

在请求位置权限之前，应用会先询问用户是否希望在外出时收到提醒通知。

3. 限定位置用途

应用会明确说明，位置仅用于本地天气显示和外出提醒通知。

4. 说明精度边界

提醒主要针对离开家或离开停留一段时间的地点。像电车等高速移动场景，定位精度会受到限制。

よくある質問

通知はどんな時に来ますか

自宅を出る時や、一定時間滞在した場所から移動する時など、忘れ物が起きやすいタイミングを対象にします。

電車の移動中も通知されますか

高速移動中は位置判定の精度に限界があるため、必ずしも狙い通りに通知されるとは限りません。

写真は自動保存されますか

撮影した写真を自動で写真ライブラリへ保存することはありません。

ゲスト利用でもクラウド同期されますか

いいえ。クラウド保存はサインイン時のみで、ゲスト利用は端末内保存です。

FAQ

When are reminders sent?

They are intended for likely forgetful moments, such as leaving home or moving away from a place after staying there for a while.

Do reminders work on trains?

Not reliably. Location precision is limited during fast transit, so train movement is not a dependable trigger.

Are photos saved automatically?

No. Captured images are not automatically saved to the device photo library.

Does guest use sync to the cloud?

No. Cloud storage is used only when the user signs in. Guest mode remains local to the device.

常见问题

提醒会在什么时候发送？

提醒主要面向容易忘带东西的时机，例如离开家时，或离开停留一段时间的地点时。

坐电车时也会提醒吗？

不一定。高速移动时的位置判断精度有限，因此电车移动并不是可靠的触发条件。

照片会自动保存吗？

不会。拍摄的图片不会自动保存到设备相册。

访客模式也会同步到云端吗？

不会。只有登录后才会进行云端保存，访客模式仅保存在本机。
